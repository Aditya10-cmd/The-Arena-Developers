🌐 Personal Portfolio Website <br><br>
📌 Project Overview<br><br>
This project is a personal portfolio website created using HTML5 and CSS3.<br>
It showcases basic information about the developer, skills, and a contact form in a clean and visually appealing layout.<br>
The website is fully responsive and uses modern CSS features like Flexbox and Grid for better structure and alignment.<br><br>
🎯 Objectives<br><br>
To understand the structure of a basic HTML webpage<br>
To apply CSS for styling, layout, and responsiveness<br>
To create a professional-looking personal portfolio<br>
To practice Flexbox and Grid for layout management<br>
